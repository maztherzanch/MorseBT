This project pretends  introduce and send a message by an app in your android cellphone by bluetooth, and the device based on arduino platform catch the message and reproduce it as a morse code by the speaker conected.

Este proyecto prentende introducir y enviar un mensage atravez de una aplicacion en tu smartphone android, cuando el dispositivo basado en la plataforma arduino reciva el mengage via bluetooth reproducira el mensaga como un mensage en codigo morse usando el buzzer que tiene conectado.
